On the SSH jumphost, configure Git: git config –global user.name “Administrator” git config –global user.email “admin@example.com” Create a new Git repository and push the contents to the Gitlab server

git init git remote add origin git@gitlab.f5.local:root/my-cnf.git git add . git commit -m “Initial commit” git push -u origin master

Return to the GitLab UI, an verify all files have been pushed into the repository.
